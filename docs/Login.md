# Página Login

## Descrição
Tela de autenticação com email/senha e mensagens de validação.

## Props utilizadas
- Nenhuma.

## Funções internas
- Valida usuário ativo na store.
- Cria sessão local com expiração.
- Informa sobre a necessidade de alimentar o seed no /system.

## Resultado esperado
- Usuário autenticado é direcionado para /app.
