# Página Gestão de Fornecedores

## Descrição
Cadastro de fornecedores com dados principais.

## Props utilizadas
- Nenhuma.

## Funções internas
- Cria fornecedores.
- Lista fornecedores cadastrados.
- Valida email obrigatório e único.
- Valida CPF/CNPJ obrigatório, único e com dígitos válidos.
- Exibe alertas imediatos de recusa no cadastro.

## Resultado esperado
- Base de fornecedores pronta para compras e contas a pagar.
