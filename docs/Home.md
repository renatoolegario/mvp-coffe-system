# Página Home (Landing Page)

## Descrição
Landing page pública com foco em divulgação do café torrado, acesso rápido ao sistema e vitrine dinâmica de produtos destacados.

## Props utilizadas
- Nenhuma.

## Funções internas
- Apresenta CTA para acesso ao login.
- Lista automaticamente os produtos de `insumos` marcados para aparecer na página inicial.
- Exibe nome, descrição, valor de venda e imagem base64 cadastrados no backoffice.

## Resultado esperado
- Visitante entende a proposta e consegue navegar para o login.
- A vitrine pública acompanha o cadastro administrativo sem depender de conteúdo hardcoded.
