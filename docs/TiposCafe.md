# Página Tipos de Café

## Descrição
Cadastro de tipos de café com rendimento, insumo base e margem de lucro para cálculo de custo.

## Props utilizadas
- Nenhuma.

## Funções internas
- Cria tipos de café.
- Lista tipos cadastrados.

## Resultado esperado
- Tipos prontos para gerar estoque diretamente na fabricação.
