# Página Tipos de Café

## Descrição

Cadastro de tipos de café com rendimento, insumo base e margem de lucro.

## Props utilizadas

- Nenhuma.

## Funções internas

- Cria tipos de café.
- Lista tipos cadastrados.
- Edita dados de tipos já cadastrados.

## Resultado esperado

- Tipos prontos para gerar estoque diretamente na fabricação.
