# Página Dashboard de Insumos

## Descrição

Painel de saldo, custo médio e valor em estoque dos insumos.

## Props utilizadas

- Nenhuma.

## Funções internas

- Busca dados consolidados no endpoint `/api/v1/dashboard/insumos`.
- Exibe saldo, custo médio e valor total em estoque retornados pelo banco de dados.

## Resultado esperado

- Visão rápida do estoque de matérias-primas com dados reais persistidos no banco.
